To install (applies to Chameleon 2.0 and higher):
	1. Close Chameleon if it is open.
	2. Copy the "Thumb" folder and the "Wallpapers" folder to the "ChameleonApp" folder in your home directory. Create the "ChameleonApp" folder if it does not already exist.
	3. Start Chameleon and the wallpaper pack will appear.
